# Security Policy

## Supported Versions

Use this section to tell people about which versions of your project are
currently being supported with security updates.

| Python | Supported           |
| ------ | ------------------- |
| <=3.9  | :x: (not supported) |
| 3.10   | :white_check_mark:  |
| 3.11   | :white_check_mark:  |
| 3.12   | :white_check_mark:  |
| 3.13   | :x: (not tested)    |
| 3.14   | :x: (not tested)    |

Only stable Flask 3.1.0+ are supported.

Only stable Flask CORS 6.0.0+ are supported.

Do only install pip packages that are explicitely described in the [requirement.txt](https://github.com/Lord-ZuzurNC/3M-MinecraftModpackMatrix/blob/main/requirements.txt).

## Reporting a Vulnerability

If you find a security vulnerability, please:

1. Open a **private security advisory** on GitHub.
2. Or email: [LordZNC@zcastle.nexus](mailto:LordZNC@zcastle.nexus)

**Please don’t disclose vulnerabilities publicly until they’ve been fixed.**
